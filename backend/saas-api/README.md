# Skincare SaaS Platform

Enterprise Multi-Tenant Commerce Experience Platform built with FastAPI.

## Technology Stack

- FastAPI
- PostgreSQL
- SQLAlchemy 2.x
- Alembic
- Pydantic v2
- uv
- Docker

## Architecture

- Modular Monolith
- Repository Pattern
- Service Layer
- Multi-Tenant
- Enterprise Security

## Development

```bash
uv sync
uv run uvicorn app.main:app --reload
```
